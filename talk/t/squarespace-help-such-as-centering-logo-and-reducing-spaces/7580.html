Hey Courtney, thanks for getting in touch! I'm sure available for this kind of work and specially the more complicated ones. If you haven't, please do read www.cregox.com/your-home

I'm not sure what you mean by "page links on either side". On Pacific I can see just page links centered right below the logo:

That is something quite simple to do. Maybe as simple as this (works in my end):

h1.site-title {
  display: table-caption;
  text-align: center;
  width: inherit;
}
#desktopNav ul {
  text-align: center;
}

But if you mean something like what happens on some Fulton sites, for instance:

Then it can be very tricky, but certainly still possible. Anything that's just tweaking the layout like this (as opposed as to adding elements) is possible.

And so, as for "reducing spaces between images", it would probably be something also as simple as tweaking on the CSS. For instance, maybe something like this could help (it would work on my site):

nav.sqs-svg-icon--list {
  a.sqs-svg-icon--wrapper {
    margin-right: 1em !important;
  }
}
